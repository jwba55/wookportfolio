개발자 리소스 공유 커뮤니티(코딩기딩)

개발자가 자신이 만든 리소스를 공유 및 판매 가능하도록 한 플랫폼입니다.

  • 프로젝트 주제 선정이유:
    팀원 중 웹 개발을 처음 접하는 분이 있어, 개발자 간의 리소스를 공유 및 판매하는 커뮤니티를 만들어보면 팀원들이 개발자로서 성장하는데 도움이 될 것 같아 주제를 선정하게 되었습니다.
  • 주요 트러블 슈팅:
    1. 자체 인증 시스템 구현의 어려움
    1. 자체 암호화 로직 구현 중 로그인
      • 사용자가 입력한 비밀번호와 함께 특수문자와 그 좌표를 DB에 저장하여 불러옴으로 해결
    2. 인프라 구축 및 배포 자동화
    1. EC2 인스턴스에 OracleDB 설치 및 설정 문제
      • Oracle에서 제공하는 Docker 이미지를 불러옴으로 해결
      • Docker Compose를 활용하여 DB가 실행된 후 애플리케이션이 실행되도록 설정
    2. Tomcat 포트 변경이 적용되지 않는 문제
      • GitHub Actions에 Server.xml을 만들어두고 Dockerfile에서 파일을 복사하도록 구성하여 해결
    3. NginX SSL 설정 문제
      • Let's Encrypt의 Certbot을 사용하여 SSL 인증서를 자동으로 발급·갱신하도록 설정하여 해결

    이 프로젝트를 진행하며 개발자로서의 실력 향상에 매우 도움될 것이라 생각했습니다.

음악 판매 및 공유 사이트

음악가가 자신의 음원을 등록해 공유 및 판매 가능하게 한 사이트 입니다.

  • 프로젝트 주제 선정이유:
    1학년 2학기 때 배운 웹 기술과 저의 관심사인 음악을 접목시켜 개인 프로젝트를 진행해보고 싶었습니다.
    또한, 음악가들이 자신의 음원을 쉽게 공유하고 판매할 수 있는 플랫폼이 부족하다고 느껴 이 주제를 선택하게 되었습니다.
  • 주요 트러블 슈팅:
    1. 주제 선정
      1. 문제
        • 랭킹시스템 구조의 어려움
      2. 해결
        • rnum을 사용해 기간 별 높은 조회수에 따라 순번을 저장하고
          저장된 순번을 SUB쿼리로 복제해서 다시 그 SUB쿼리를 조회하는 구조로 해결
    2. 첫 파일 업로드에 대한 어려움
      1. 문제
        • 음원 파일만 업로드 되어야하는 문제
      2. 해결
        • 파일 업로드 시 확장자를 구분하여 확장자가 mp3, ogg일 경우에만 업로드 되도록 해결

    C:import를 사용하여 네비게이션바, footer, 활동 영상, 플레이리스트페이지를 각각 불러와서 보여줌.
    JavaScript를 사용하여 활동 영상을 3초 단위로 자동 슬라이드.

  • 회원가입/로그인 페이지:

    Session을 사용하여 로그인 여부를 확인하고 회원에게는 로그아웃/파이페이지,
    비회원에게는 회원가입/로그인 버튼이 보이게 함.

  • 곡 조회 페이지:

    등록된 모든 곡 정보를 불러와서 보여줌.

  • 곡 등록 페이지:

    Session에 저장된 role 값에 따라 일반 회원은 무료 공유만 가능하고 파트너 회원은 유료 판매도 가능하게 함.
    파일 명과 UUID를 저장하고 파일의 확장자를 구분하여 음성 파일만 업로드 가능하게함.

  • 순위 페이지:

    Component에 작성한 Scheduler를 사용하여 일간,주간,월간 별로 조회수를 초기화하고 랭킹을 갱신함.

  • 상품 상세 페이지:

    상품의 상세 정보 및 등록된 파일을 확인하고, 리뷰를 남기거나 구매할 수 있도록 함.

  • 플레이리스트 페이지:

    저장되어 있는 모든 플레이리스트를 불러와서 보여줌.
    멜론 처럼 각각의 다른 회원들이 설정한 플레이리스트를 확인하고 불러올 수 있도록 함.

  • 플레이리스트 등록 페이지:

    플레이리스트 명을 입력하고 저장 버튼을 누르면 플레이리스트가 등록됨.
    생성된 플레이리스트에 내가 선택한 곡을 담을 수 있게 함.(장바구니 같은 형태)

  • 이벤트 페이지:

    관리자가 작성한 이벤트 및 공지사항을 확인할 수 있도록 함.
